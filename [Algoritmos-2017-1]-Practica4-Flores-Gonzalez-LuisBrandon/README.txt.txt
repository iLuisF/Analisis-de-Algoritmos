-------Manera de ejecutar el programa-------

El método main se encuentra en el archivo Dijkstra.java, por lo que es suficiente compilar y ejecutar este archivo.
Además las estructuras de datos se encuentran en un solo folder.

--------Explicación de como construir la gráfica--------

1. Se crea un objeto grafica, con parametro false. Ya que es no dirigida.
2. A tal objeto se le agregan aristas, donde el primer parametro es el identificador 
del primer vertice y el segundo parametro es el identificador del segundo vertice.
3. Se crea un objeto Dijkstra  para poder aplicar el algoritmo.
4. Se crea una vertice fuente.
5. Se guarda el resultado del algoritmo en un map.
6. Se muestra el resultado.

En Dijkstra.java en el método main se encuentra un ejemplo sencillo pero bastante ilustrativo.
