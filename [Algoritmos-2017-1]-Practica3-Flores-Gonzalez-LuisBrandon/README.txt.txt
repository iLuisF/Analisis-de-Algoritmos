-------Manera de ejecutar el programa-------

El método main se encuentra en el archivo Prim.java, por lo que es suficiente compilar y ejecutar este archivo.
Además las estructuras de datos se encuentran en un solo folder.

--------Explicación de como construir la gráfica--------

1. Se crea un objeto grafica, con parametro false. Ya que es no dirigida.
2. A tal objeto se le agregan aristas, donde el primer parametro es el identificador 
del primer vertice y el segundo parametro es el identificador del segundo parametro.
3. Se crea un objeto Prim para poder aplicar el algoritmo.
4. Se crea una lista ligada de aristas, donde se le asigna la aplicación de Prim con la gráfica creada.
5. Finalmente se recorre la lista ligada para imprimr el resultado de apligar el algoritmo.

En Prim.java en el método main se encuentra un ejemplo sencillo pero bastante ilustrativo.
